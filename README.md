#TRADEnova 📈

TRADEnova is an AI-powered stock price prediction system that uses LSTM (Long Short-Term Memory) networks to analyze historical data and forecast future stock price movements.


 🚀 Features

 📊 Stock price prediction using LSTM
 📈 Visualization of predicted vs actual prices
 🌐 Interactive frontend built with React
 ⚡ Fast and responsive UI

 🛠️ Tech Stack

 Frontend: React (Vite), TypeScript
 Backend: Node.js
 Machine Learning: LSTM (Deep Learning)


 📂 Project Structure

 client/ → Frontend UI
 server/ → Backend APIs
 shared/ → Shared logic


▶️ How to Run

cd client
npm install
npm run dev

📌 Future Improvements

 Real-time stock API integration
 More advanced models (GRU, Transformers)
